# CNN-Based Inverse Design for G-HMWG Antenna

This repository contains the raw training code used for the CNN-based inverse design
of the ultra-wideband Grooved Half-Mode Waveguide (G-HMWG) antenna.

## Reproducibility

- Random seed fixed to 42
- Compatible with Python 3.9+

## Requirements

Install dependencies:

    pip install -r requirements.txt

## How to Run

    python train.py

## Dataset

Please place your dataset in the same directory or update the dataset path inside train.py.

## Notes

This code corresponds to the model used in the submitted manuscript.
